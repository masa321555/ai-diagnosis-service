# LP設計書

## 概要

AIキャリア診断サービス「5問でわかる、あなたのキャリア」のランディングページ設計書。

### 目的
- サービスの価値を伝え、ユーザーに診断を開始してもらう
- 20%以上のCTA押下率を目指す

### ターゲット
- 就活中の学生
- 転職を考えている社会人
- キャリアに悩む20〜30代

## セクション詳細設計

### 1. ヒーローセクション（HeroSection）

#### 目的
ファーストビューでサービスの価値を伝え、興味を喚起する

#### コンテンツ

| 要素 | 内容 | 備考 |
|-----|------|------|
| キャッチコピー | 5問でわかる、あなたのキャリア | h1タグ、最大フォントサイズ |
| サブコピー | AIがあなたに最適なキャリアロードマップを提案します | h2またはp、キャッチの補足 |
| CTAボタン | 無料で診断を始める | プライマリボタン、目立つ配色 |
| 補足テキスト | 登録不要・3分で完了 | ボタン下に小さく表示 |
| 背景 | グラデーションまたは抽象的なビジュアル | 信頼感・先進性を演出 |

#### デザイン仕様

```
背景: linear-gradient(135deg, #667eea 0%, #764ba2 100%)
文字色: #FFFFFF
CTAボタン: 
  - 背景: #FFFFFF
  - 文字色: #667eea
  - border-radius: 50px
  - padding: 16px 48px
  - hover: box-shadow追加
```

#### レスポンシブ対応

| ブレークポイント | フォントサイズ(h1) | 高さ |
|----------------|-------------------|-----|
| PC (md以上) | 3.5rem | 100vh |
| タブレット (sm) | 2.5rem | 80vh |
| スマホ (xs) | 2rem | auto (padding調整) |

---

### 2. 特徴セクション（FeatureSection）

#### 目的
サービスの3つの特徴を簡潔に伝える

#### コンテンツ

| 特徴 | アイコン | タイトル | 説明 |
|-----|---------|---------|------|
| 1 | AccessTime | たった5問・3分で完了 | 忙しいあなたでも隙間時間でOK。シンプルな質問に答えるだけです。 |
| 2 | Psychology | AIが深く分析 | 最新のAI技術があなたの回答を多角的に分析します。 |
| 3 | PersonPin | パーソナライズされた提案 | あなただけのキャリアロードマップを作成します。 |

#### デザイン仕様

```
セクション背景: #FAFAFA
カード:
  - 背景: #FFFFFF
  - border-radius: 16px
  - box-shadow: 0 4px 20px rgba(0,0,0,0.08)
  - padding: 32px
アイコン:
  - サイズ: 48px
  - 色: primary (#667eea)
  - 背景: rgba(102, 126, 234, 0.1)
  - padding: 16px
  - border-radius: 50%
```

#### レイアウト

| ブレークポイント | レイアウト |
|----------------|-----------|
| PC (md以上) | 3カラム横並び |
| タブレット (sm) | 2カラム |
| スマホ (xs) | 1カラム縦積み |

---

### 3. ステップセクション（StepSection）

#### 目的
診断の流れを3ステップで視覚的に説明する

#### コンテンツ

| ステップ | アイコン | タイトル | 説明 |
|---------|---------|---------|------|
| Step 1 | QuestionAnswer | 5つの質問に回答 | 性格や価値観に関するシンプルな質問に答えてください |
| Step 2 | AutoAwesome | AIが即座に分析 | 回答をAIがリアルタイムで分析します |
| Step 3 | Assessment | キャリアマップ表示 | あなた専用のキャリアロードマップが完成します |

#### デザイン仕様

```
セクション背景: #FFFFFF
ステップ番号:
  - サイズ: 64px
  - 背景: primary gradient
  - 文字色: #FFFFFF
  - border-radius: 50%
コネクタライン:
  - 色: rgba(102, 126, 234, 0.3)
  - スタイル: dashed
  - PC: 横線
  - スマホ: 縦線
```

#### レイアウト

| ブレークポイント | レイアウト |
|----------------|-----------|
| PC (md以上) | 横並び、矢印でつなぐ |
| スマホ (xs) | 縦並び、縦線でつなぐ |

